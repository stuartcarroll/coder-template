# Technology Stack

## Infrastructure & Configuration
- **Terraform**: Infrastructure as Code for Coder templates
- **Docker**: Container runtime and image building
- **Coder**: Cloud development environment platform

## Container Base
- **OS**: Ubuntu 24.04 LTS
- **Shell**: Bash

## Development Tools
- **Node.js**: Version 22 LTS
- **Python**: Version 3.12 with pipx
- **GitHub CLI**: For repository management
- **VS Code Server**: Browser-based IDE

## AI & Assistance Tools
- **Claude Code CLI**: AI coding assistant
- **SuperClaude**: Enhanced Claude features with web search
- **MCP Servers**: Model Context Protocol servers for various integrations

## Supported Development Patterns
1. Laravel Standard (PHP)
2. Python API (FastAPI/Flask)
3. Python Data Science
4. Node.js Full-Stack (React/Vue)

## Service Integrations
- PostgreSQL
- MySQL
- Redis
- MinIO
