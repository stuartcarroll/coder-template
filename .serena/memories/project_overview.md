# Claude Development Environment - Coder Template

## Project Purpose
This is a Terraform-based Coder template that creates fully-configured development environments with automatic Claude Code authentication, file-based secrets management, dynamic MCP integration, and pre-configured Claude context awareness.

## Repository Information
- **GitHub Repository**: https://github.com/stuartcarroll/claude-dev.git
- **Local Path**: C:\Users\StuCarroll\coder template\claude-dev
- **Type**: Infrastructure as Code (Terraform) for Coder workspace templates

## Key Features
1. **Advanced Secrets Management**: File-based integration with automatic fallback to environment variables
2. **Intelligent Claude Code Integration**: Zero-touch authentication with pre-loaded context
3. **Development Tools & Services**: VS Code Server, GitHub CLI, SuperClaude, and more
4. **Smart Workspace Features**: Graceful degradation and dynamic status reporting
5. **Robust Container Environment**: Ubuntu 24.04 LTS with Docker support

## Target Users
Developers who need a fully-configured cloud development environment with integrated AI assistance through Claude Code.
